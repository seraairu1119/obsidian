# テーマで行かせる強み

私の一つ目の強みは、AIを活用して業務課題の改善策を形にできる力です。

長期インターンでは、業務効率化を目的として、生成AIを活用した仕組みづくりに取り組みました。

課題は、営業リスト作成の精度に改善の余地があったことと、オウンドメディア作成に時間がかかっていたことです。

営業リストでは、住所や電話番号などの企業情報が不足していることで、条件に合う企業を十分に抽出できず、リスト件数が少なくなってしまうことがありました。

また、オウンドメディア作成では、記事の構成案や本文作成に時間がかかっており、効率化の余地がありました。

そこで私は、営業リストの住所や電話番号などの欠損情報をAIで補完するためのSkillsを作成しました。また、オウンドメディアを自動生成できるSkillsも作成し、記事作成業務の効率化に取り組みました。単にAIを使うのではなく、どの作業をAIに任せ、どこを人が確認すべきかを意識しながら仕組みに落とし込むことを大切にしました。

その結果、企業情報の補完や記事作成にかかる作業負担を減らし、インターン生のタスクのこなせる量が大幅に増えました。また現在も、OpenCrawlやElme’s Agentなどの新しいAI関連技術について調査し、業務効率化に活用できないかを考えています。

この経験から、AIは導入すること自体が目的ではなく、業務上の課題を解決する手段として活用することが重要だと学びました。本テーマでも、お客様の課題を整理し、AIを活用したナレッジサービスの提案や検証を行う場面で、この強みを活かせると考えています。

**私の二つ目の強み**は、相手の要望を汲み取り、それを具体的な形にできる力です。

私は小さい頃から趣味としてイラスト制作を続けており、高校時代からはスキル販売サービスを通じて依頼を受けるようになりました。

その目的は、依頼者が思い描いているイメージを作品として表現することでした。

一方で、依頼者の要望が必ずしも明確に言語化されているとは限らない点が課題でした。例えば、「優しい雰囲気にしたい」「かっこよくしたい」といった抽象的な要望も多く、そのままでは完成イメージにずれが生じる可能性がありました。

そこで私は、参考画像を見せていただいたり、色味や構図、表情などを細かく確認したりすることで、依頼者が本当に求めている内容を整理しました。また、ラフ画の段階で認識合わせを行い、修正を重ねながら完成度を高めることを意識しました。

その結果、多くの依頼者に満足していただくことができ、高校時代から累計で約20万円の収益化につなげることができました。

この経験から、相手の要望を正確に理解し、曖昧な内容を具体的な形へ落とし込む力を身につけました。本テーマでも、顧客や現場の課題を整理し、提案や検証内容に落とし込む場面で活かせると考えています。

**私の三つ目の強み**は、立場や考え方の異なる人たちと協力しながら、目標に向かって取り組める環境を作る力です。

私は6人組のアカペラチームのリーダーとして、大会出場を目標に活動していました。

その中での課題は、メンバーごとに練習への考え方や参加状況が異なっていたことです。細部までこだわりたい人もいれば、楽しさを重視したい人、研究や授業の関係で十分に練習へ参加できない人もいました。そのため、チームとして同じ方向を向くことが難しい状況でした。

そこで私は、各メンバーと個別に話し合い、それぞれが何を重視しているのかを理解することから始めました。話し合った結果、録音を活用して課題を客観的に共有し、改善策を各自が実践できる形に整理しました。さらに、欠席者にも録音データや改善内容を共有し、練習状況が伝わるようにしました。また、練習中には必ず休憩時間を設け、雑談を通してコミュニケーションが取りやすい雰囲気づくりも意識しました。

その結果、メンバー全員が前向きに練習へ取り組めるようになり、応募約900組のうち100組が進出する東日本三次予選まで進むことができました。

この経験から、相手の状況や価値観を理解した上で進め方を調整し、チーム全体をまとめる力を身につけました。本テーマでも、お客様やチームメンバーなど様々な立場の人と協力しながら、課題解決を進める場面で活かせると考えています。
