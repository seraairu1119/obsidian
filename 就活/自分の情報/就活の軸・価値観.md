# 就活の軸・価値観



## Core Positioning



- 志望軸: 顧客課題を整理し、納得感のある提案で解決するSE・ITソリューション人材を目指す。

- 強み: データ分析、課題整理、顧客ヒアリング、チームでの議論、提案改善、実行までやり切る姿勢。

- 仕事観: 信頼できる仲間と技術を高め合い、チームでプロジェクトに取り組みたい。

- 価値観: 一人の視点に閉じず、異なる視点を整理して本質的な課題解決につなげる。

- 将来関心: ビジネスを構想し、新たな価値を社会に届ける仕事に関わりたい。



## User's Self-Analysis Core



The user's central identity:



> チームで課題に向き合い、意見を出し合いながら改善を積み重ね、成果につなげることにやりがいを感じる人。



Use the following as the stable backbone.



- 価値観: 仲間と同じ目標に向かって試行錯誤し、努力や改善の積み重ねが成果として表れることにやりがいを感じる。

- 強み: 分からないことや複雑な課題に対して、情報を調べ、理由や根拠を整理し、周囲の意見も踏まえて納得感のある方向性を作れる。

- コミュニケーション: 相手の意見を否定せずに聞き、なぜそう考えたのか理由まで確認する。自分の意見も理由とセットで伝える。

- チーム観: 意見を出し合える雰囲気を重視する。単なる作業分担ではなく、各自の視点を持ち寄り改善していくチームに惹かれる。

- 弱み: より良い案を探そうとして、自分が腑に落ちるまで考えすぎ、判断に時間がかかることがある。

- 改善行動: 調べる範囲や期限を区切り、仮説段階でも理由を添えて周囲に共有し、比較検討しながら前に進める。

- 合う環境: チームで意見を出し合い、目的や理由を共有しながら改善を重ねられる環境。

- 合わない環境: 一人で完結する仕事、意見を出しにくい雰囲気、理由が共有されず指示だけで進む環境、個人競争が強い環境、改善の余地が少ない反復作業。

- スピード感: スピード重視自体は問題ない。目的や理由が共有されていれば、速く調べて提案しながら動ける。



## Interview Axis Draft



Use this when asked about 就活の軸 or desired environment:



> 私は、チームで意見を出し合いながら、改善を積み重ねて成果につなげられる環境で働きたいと考えています。ただ作業を分担するだけでなく、それぞれが気づいたことや改善案を出し合い、少しずつ完成度を高めていく過程にやりがいを感じるからです。スピード感のある環境でも、目的や理由が共有されていれば、自分で調べて提案しながら前に進められると考えています。



## Weakness Draft



Use this when asked about weakness:



> 私の課題は、より良い案を探そうとして、判断に時間がかかることがある点です。何かを決める際に、後からもっと良い方法が見つかるのではないかと考え、自分が腑に落ちるまで調べたり比較したりしてしまうことがあります。一方で、仕事では限られた時間の中で判断することも重要だと考えています。そのため現在は、調べる時間や判断の期限を決め、現時点で分かっている情報をもとに仮説として周囲に共有することを意識しています。



### Strength And Weakness



長所: 状況に応じて最適な提案を行い、周囲と協働しながら価値を高められる点。



短所: 自分の中で納得してから進めたいという意識が強く、自分のペースで進めてしまう点。



理由: 人と関わる中で、相手の状況や意図を踏まえて提案の内容や進め方を工夫することで、納得感のある意思決定や成果につながると実感してきたため。一方で、自分の中で考えを整理し納得した上で進めたいという意識が強く、自分のペースで物事を進めてしまう傾向があると感じているため。



エピソード: 日常の人間関係や活動において、何かを決める際には相手の状況や意見を踏まえた上で提案を行い、より良い形を一緒に作ることを意識してきた。例えば、アカペラの活動ではメンバーの状況に応じて練習方法や進め方を提案し、チームとしてのまとまりを高めてきた。一方で、自分の中で納得してから行動したいという意識が強く、周囲との進め方にずれが生じることもあったため、現在は早い段階で自分の考えを共有し、周囲とすり合わせながら進めることを意識している。



### Job-Hunting Axis



軸: 人と近い距離で関わりながら課題解決を行い、納得感のある意思決定を支える仕事。



理由: 人と議論しながら考えることで、自分一人では出せないアイデアが生まれ、より多くの人が納得できる最適な答えにたどり着けると感じたため。また、自分の考えが他者の意思決定に影響を与え、価値として発揮されることにやりがいを感じているため。



エピソード: インターンにおいて、データを用いた営業リストの作成を通じて顧客の課題解決に取り組んだ。その中で、チーム内で「どのような営業リストを作成すべきか」「効率的に進めるためにどのような開発を行うべきか」といった議論を重ねながら進めた。特に、メンバーそれぞれの視点を取り入れることで、自分一人では思いつかない発想が生まれ、より精度の高い提案につながった。その結果、顧客に提供した営業リストによってアポイント数が20％向上し、自分たちの提案が実際の成果に結びつくことを実感した。この経験から、人と近い距離で関わりながら課題解決を行い、意思決定を支える仕事に魅力を感じるようになった。
